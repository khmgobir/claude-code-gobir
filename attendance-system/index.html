Google Cloud setup guide
  1. Go to console.cloud.google.com and create a project.
  2. Enable Google Sheets API.
  3. Go to Credentials → Create → OAuth 2.0 Client ID.
  4. Application type: Web application. Add your site URL to Authorized JS origins.
  5. Copy the Client ID and paste it above.
  6. Create a blank Google Spreadsheet for attendance data and paste its ID.
  7. Click Save Settings then Connect & Sign In.
